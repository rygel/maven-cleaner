# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities by opening a [GitHub Security Advisory](https://github.com/rygel/maven-cleaner/security/advisories/new) (private disclosure). You will receive a response within 7 days. If the issue is confirmed, a patch will be released as soon as possible.
